Requirements

It requires a GPU card on the system and Linux 64 bits is preferred.

  • python 2.7 [https://github.com/python/cpython/tree/2.7]
  • gputil [https://github.com/anderskm/gputil]
  • scipy [https://github.com/scipy/scipy]
  • pyDOE [https://pythonhosted.org/pyDOE/]
  • edward 1.3.5 [https://github.com/blei-lab/edward]
  • tensorflow 1.8 [https://github.com/tensorflow/tensorflow]

Methodology

We leverage Mean-field Variational Inference inferring a Bayesian neural network to learn the Koopman eigenfunctions, eigenvalues, for a continuous nonlinear dynamical systems as \(\dot{x} = F(x)\).

For detailed information, please refer to our preprint [https://arxiv.org/abs/1906.03663].
