samplingX_Xdot(F, range_of_X, num_samples_each_dim=10)¶

Sampling X and Xdot given ground true governing equation using LatinHyperCubic from pyDOE

In default, we use LatinHyperCubic, pyDOE.lhs() to generate the sampling for the design of experiments (DOE) in the hyper-cubic on \([0, 1]\). Then we scale it back according to range_of_X. In default, we use center criterion and only one iteration. Then we call function F to get the time derivative \(\dot{x}\) and Jacobian. However, in the most recent code, we don’t need Jacobian. So one can simply return 0 array.

Parameters:
  • F (function) – a Python function that calls the state \(x\) and return the time derivative \(\dot{x}\) and Jacobian. However Jacobian is not used any more so one can just leave it as zero array.
  • range_of_X (numpy.ndarray) – The range for each component to get sampled from.
  • num_samples_each_dim (int) – Number of samples for each component. Default value is 10.
