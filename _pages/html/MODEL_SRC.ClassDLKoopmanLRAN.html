MODEL_SRC.ClassDLKoopmanLRAN module¶

Class of Continuous time Deep Learning Models for Koopman Operator with LRAN

class MODEL_SRC.ClassDLKoopmanLRAN.ClassDLKoopmanLRAN(configDict, edward_configDict=None, gpu_percentage=0.25)¶

Bases: MODEL_SRC.ClassDLKoopmanCont.ClassDLKoopmanCont

Class of LRAN

Simply using multiple steps in to the future, i.e., T, for prediction and minimize error. Moreover, data is split into training and testing by their sequential order.

Parameters:
  • configDict (dict) – dictionary of the configuration on deterministic side for LRAN
  • edward_configDict (dict) – dictionary of configuration on Bayesian DL model.
  • gpu_percentage (float) – percentage for GPU memory usage.
T¶

int – number of steps of future to look forward to see.

dt¶

float – time interval between snapshots since I am doing LRAN, I need dt since the reconstruction requires time.

numpy.ndarray – raw trajectory with no hankerization and normalization.

hankel_data¶

numpy.ndarray – hankerized data with T.
