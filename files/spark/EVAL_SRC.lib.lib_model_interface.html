class EVAL_SRC.lib.lib_model_interface.ClassFNN_edward(sess, layer_structure_list, act_fun_str, prefix, mode, LRAN_T, SVD_DMD)¶

Bases: object

Class of Feedforward neural network given topology, structure, activation function, and recurrent or not, SVD_DMD or not, to replicate the model we saved.

Note

We did this simply because there is no good save_model functionality in edward.

Parameters:
  • sess (class) – the session we used in tensorflow.
  • layer_structure_list (list) – the layer structure we used in constructing the Feedforward neural network.
  • act_fun_str (str) – the name of the activation function to use.
  • prefix (str) – the prefix used in the weights and biases parameters, either dec or enc.
  • mode (str) – the type of ADVI to use, either MAPSimple or ADVInoARD, ADVIARD.
  • LRAN_T (int) – the look forward time window length.
  • SVD_DMD (bool) – flag on whether or not SVD DMD is enabled.
